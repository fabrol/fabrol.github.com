To Done Lists

Live Source Code

A visualization tool for Microsoft To-Do that helps users understand their productivity patterns by displaying completed tasks over time.

Features

  • Fully client-side, no one can see your data
  • Timeline visualization of completed tasks
  • Aggregated views up to one year
  • Secure authentication via Microsoft Graph API

Technologies Used

  • Next.js
  • TypeScript
  • Microsoft Graph API
  • Vercel for deployment
