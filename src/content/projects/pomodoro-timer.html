Pomodoro Timer

Live Source Code

A minimalist, animated pomodoro timer built to help with focus and productivity. This project was my way of getting back into frontend development while creating something useful for my own workflow.

Features

  • Clean, minimalist interface
  • Smooth animations using Framer Motion
  • Configurable work/break intervals
  • Desktop notifications

Technologies Used

  • React
  • TypeScript
  • Framer Motion for animations
